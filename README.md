# cdlpm
### a ts-node template creator

## install

```shell
yarn global add cdlpm
```

```shell
npm install cdlpm -g
```
## usage
```shell
cdlpm create <dirname> [template]
```
will create a dir named 'dirname'

if template is empty it will be 'basic'
by default
